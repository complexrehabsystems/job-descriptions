Senior Fullstack Developer

Skills Overview

Required: >5 yrs experience, C# 6, Microsoft SQL Server, ASP.NET WebApi or ASP.NET Core

Desired: Xamarin, EF6 code first migrations, good UX/UI sensibilities, C/C++, 3D graphics, NUnit, RhinoMocks, AutoFac, MVVM architectures

Complex Rehab Systems is seeking a full time or part time Senior Fullstack Developer (C#/Xamarin/SQL Server) who will contribute to an exciting 3D scanning healthcare application for clinicians working in Complex Rehab.

Come join a small team of skilled C# developers working on an exciting 3D scanning healthcare product, where you will enjoy working remotely and flexible hours.

The application comprises a RESTful API written in .net core, a native Xamarin + UWP frontend, and we have plans to build a modern web portal. There are components written in C#, C/C++, and HTML/CSS/javascript. You will have significant opportunities to take ownership of large projects, and contribute to the technical direction of the company.
